### 
A simplistic State Machine using PlantUML and LaTeX syntax.

### Prerequisites
1. LaTeX  
2. vimtex
3. PDF Viewer 
   

### Installation 
Copy the text below and paste into a shell.

```bash
cd $GIT_HOME && git clone https://github.com/traap/plantuml.git
```
